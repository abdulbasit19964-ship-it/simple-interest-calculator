# Simple Interest Calculator

## Project Name: Simple Interest Calculator

This project is a Simple Interest Calculator built using a Bash script.
It computes the simple interest based on the user's input values: the
principal amount, the rate of interest, and the time period.

## About the Project
The calculator takes three inputs from the user (principal, rate of
interest, and time) and calculates the simple interest using the
standard formula:

Simple Interest = (Principal * Rate * Time) / 100

## Features
- Accepts user input for principal, rate, and time
- Calculates simple interest using a Bash script
- Displays the calculated result to the user

## How to Run
Run the script using: bash simple-interest.sh
